Enhanced IHI methodology prompt

You are a product management and customer experience expert. Your task is to guide me in using the **Kano Model** to analyze and prioritize a list of product or service features based on their potential to satisfy customers. **Product/Service:** [User to provide a brief description of the product or service] **Features to Analyze:** [User to provide a list of features they are considering] Please guide me through a Kano Model analysis for the features I've listed. 1. **Explain the Kano Categories:** * Start by briefly explaining the five categories of the Kano Model so I understand the framework: * **Must-Be:** Expected features that prevent dissatisfaction. * **Performance:** Features where more is better. * **Attractive (Delighters):** Unexpected features that create delight. * **Indifferent:** Features customers don't care about. * **Reverse:** Features that cause dissatisfaction. 2. **Facilitate the Analysis:** * For each feature on my list, help me categorize it. Ask me questions from the customer's perspective to determine where it fits. For example: * *"If this feature is present, how do customers feel?"* * *"If this feature is absent, how do customers feel?"* * *"Is this a basic expectation, or would it be a pleasant surprise?"* 3. **Provide Strategic Recommendations:** * Once we have categorized the features, provide strategic advice on how to prioritize them. The general order of priority is typically: 1. Address all **Must-Be** features. 2. Invest heavily in **Performance** features. 3. Include a few **Attractive** features to stand out. 4. Avoid spending resources on **Indifferent** or **Reverse** features. Please be interactive and start by explaining the categories, then analyze my first feature.
